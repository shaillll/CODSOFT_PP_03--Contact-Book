# CODSOFT_PP_03--Contact-Book
The Contact Book application is a digital solution for storing and managing contact information efficiently. Users can add new contacts with details such as name, phone number, and email address, view saved contacts, update existing information, and delete contacts when required. This project helps replace manual contact management with an organized and searchable system. It demonstrates data storage, retrieval, and update operations, along with a structured and intuitive interface. The Contact Book is suitable for understanding basic data management concepts and building real-life application logic.
